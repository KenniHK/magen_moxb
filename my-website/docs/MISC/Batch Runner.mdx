## Cara Menjalankan Batch menggunakan web Batch Runner

1. Akses web Batch Runner (http://10.43.6.177:55205/runner-batch/index.html).
![BATCH RUNNER 1](/img/dokumen_magenta/batch-runner-1.png)

2. Cari batch yang ingin dijalankan dan tahan tombol 'â–¶ Run' hingga berwarna cokelat untuk menjalankan batch.
![BATCH RUNNER 2](/img/dokumen_magenta/batch-runner-2.png)

    Jika batch membutuhkan file input, upload file terlebih dahulu untuk menjalankan batch. Tanpa file, batch tidak akan berjalan.
    ![BATCH RUNNER 3](/img/dokumen_magenta/batch-runner-3.png)

    Tampilan berikut adalah tampilan saat batch berjalan.
    ![BATCH RUNNER 4](/img/dokumen_magenta/batch-runner-4.png)

3. Setelah proses menjalankan batch selesai, log akan muncul di bagian kiri bawah seperti gambar berikut. Log akan menghilang dengan sendirinya setelah 5 detik.
![BATCH RUNNER 5](/img/dokumen_magenta/batch-runner-5.png)

    Klik atau scroll log akan memperbesar tampilannya dan menahan log agar tidak menghilang, sementara memindahkan cursor dari log akan mengecilkan log kembali.

4. Setelah log menghilang, log masih bisa dilihat pada log history.
![BATCH RUNNER 6](/img/dokumen_magenta/batch-runner-6.png)

    Log history dapat dilebarkan dengan menarik area putih di pojok kanan bawah. Jika log menghasilkan file, file dapat didownload pada log history.
    ![BATCH RUNNER 7](/img/dokumen_magenta/batch-runner-7.png)

## Cara Menjalankan Batch secara manual
1. Akses WinSCP dan login ke site batch.
![BATCH RUNNER 8](/img/dokumen_magenta/batch-runner-8.png)

    Jika session belum tersimpan, klik 'New Site' dan isi informasi sebagai berikut, lalu login.
    ![BATCH RUNNER 9](/img/dokumen_magenta/batch-runner-9.png)

    | File protocol | SFTP        |
    |---------------|-------------|
    | Host name     | 10.43.5.165 |
    | Port number   | 22          |
    | User name     | appmagen    |
    | Password      | Bcabca01    |

2. Setelah login, masuk ke folder "batch".
![BATCH RUNNER 10](/img/dokumen_magenta/batch-runner-10.png)

3. Cari batch yang ingin dijalankan dan masuk ke folder tersebut.
![BATCH RUNNER 11](/img/dokumen_magenta/batch-runner-11.png)

    Cari file .bat, lalu klik ikon open terminal.
    ![BATCH RUNNER 12](/img/dokumen_magenta/batch-runner-12.png)

    Ketik "./[nama file].bat" yang ingin dijalankan pada input command, lalu klik 'Execute' atau tekan 'Enter' pada keyboard. Contoh pada gambar: "./EDC.PRODUCTION.bat"
    ![BATCH RUNNER 13](/img/dokumen_magenta/batch-runner-13.png)

    Hasil menjalankan batch akan langsung terlihat pada terminal.
    ![BATCH RUNNER 14](/img/dokumen_magenta/batch-runner-14.png)

    Jika batch menghasilkan file, file dapat dilihat pada folder result/output/out di folder "batch".
    ![BATCH RUNNER 15](/img/dokumen_magenta/batch-runner-15.png)
